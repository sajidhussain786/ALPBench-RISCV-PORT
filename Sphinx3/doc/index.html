CMU Open Source Sphinx-3 Decoder

Sphinx Speech Group
School of Computer Science
Carnegie Mellon University
Pittsburgh, PA 15213

This is the current working Sphinx-3, s3.3 code base. It contains a front end that allows for stream processing. However s3.3 does not split nor parse the incoming audio data. It assumes it has been segmented already. The provided set of models requires input signal to be sampled at 16kHz, and requires audio in raw PCM format.
